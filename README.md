# Veloquent

Veloquent is an open-source Backend-as-a-Service (BaaS) built on Laravel. It provides authentication, database, storage, real-time capabilities, and multi-tenancy out of the box. This is a scaffold off the project.

For more information visit [Core Repository](https://github.com/kevintherm/veloquent)
