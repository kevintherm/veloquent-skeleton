# Veloquent Skeleton

Veloquent is an open-source Backend-as-a-Service (BaaS) built on Laravel. It provides authentication, database management, storage, real-time capabilities, and multi-tenancy out of the box.

This repository serves as the official skeleton for starting new projects with Veloquent.

## Quick Start

You can create a new Veloquent project using Composer:

```bash
composer create-project veloquent/veloquent your-project-name
```

## Features

- **Multi-tenant by design**: Build SaaS applications easily.
- **Dynamic Collections**: Manage your database schema through a powerful API or Admin Panel.
- **Real-time**: Built-in support for real-time events and data syncing.
- **Secure**: Modern authentication with token support and flexible permissions.
